 Ashika Academy Website

This is the official website for **Ashika Academy**, providing beauty services and courses including Nails, Beauty Courses, Full Body Makeup, and Lashes. The website is fully responsive and works on both desktop and mobile devices.

## Features

- Modern and luxurious black & gold design.
- Services section with detailed descriptions and icons.
- Pricing section for each service.
- Booking form for customer appointments.
- Customer care contact section.
- Responsive layout for mobile and desktop.

## Technologies Used

- HTML5
- CSS3
- Font Awesome icons

## How to Use

1. Clone or download the repository.
2. Open `index.html` in your web browser to view the website.
3. Optional: Deploy on GitHub Pages, Netlify, or Vercel for live access.
